## Compile the Kotlin Example

```console
./bld compile
```

## Run the Example

```console
./bld run
```

## Run the Tests

```console
./bld test
```

## Requirements

- A Kotlin compiler must be [installed](https://github.com/rife2/bld-kotlin?tab=readme-ov-file#kotlin-compiler-requirement).
